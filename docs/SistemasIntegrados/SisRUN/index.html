SisRUN

A Gerencianet fechou uma nova parceria com a SisRUN, sistema completo para assessorias esportivas. A plataforma oferece serviços como disponibilização de planilha de treinos, controle financeiro, aplicativos, calendário de provas, entre outras funcionalidades.

Agora, também será possível usar a Gerencianet como módulo de pagamentos para integração para fazer cobranças de forma rápida e descomplicada.


Veja como fazer a integração:#

Na sua conta Gerencianet, acesse o menu API > Minhas Aplicações;

  1. Clique no botão Nova Aplicação;

  2. Insira o nome que deseja utilizar e clique no botão Criar Nova Aplicação;

  3. Copie os códigos exibidos nos campos Client_Id e Client_Secret. Eles serão utilizados na conta SisRUN. Utilize os dados disponíveis na aba “Produção”.

Em seguida, acesse sua conta SisRUN e siga os passos abaixo:

  1. Acesse o menu Administrador > Configurações Financeiro;

  2. Escolha a opção Gerencianet;

  3. Informe seu Client_Id e Client_Secret;


Pronto. A integração está completa.

Depois disso, é só começar a gerar suas cobranças da Gerencianet diretamente pelo site SisRUN. É possível escolher o envio do boleto ou link de pagamento em que o cliente escolhe pagar com boleto ou cartão de crédito. Para saber mais sobre as soluções e tarifas, clique aqui.


Em caso de dúvidas, entre em contato com a Gerencianet ou SisRUN.
